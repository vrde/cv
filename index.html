Isoverse

view project

Isoverse is the new publishing platform for the visual arts, fully integrated with your workflow. With Isoverse you save files on your computer and you're right online: no upload, no hosting, no code. When you sign up, Isoverse asks you to create a new folder in your Dropbox. Open that folder, drop your images and texts there and it's done: your work is automatically visible online. Tech stack: Python (Django), RabbitMQ and Celery.

Manana

view project

Manana is a simple, bandwidth-saving way to enjoy your favourite articles without unnecessary distractions. You can read them anytime, tag and share them with your friends. Freelance developer for embedded devices

  • November 2010–October 2012
  • Milan (Italy)

I worked with two other engineers on a cutting edge technology for street lighting. The smart street lights we built were equipped with a microcontroller and were able to communicate with a central router. My role was to write the Python code to run in the router. The router was programmed to: turn on/off the lights, dim the light intensity, collect information about the condition of the lights installed, compress and send the collected data to a remote server, and display on a screen the state of the lights.

The system was integrated also with the legacy controller provided by the client, and it was able to react to emergency situations. The system was able to communicate via RS485 and XBee, using the MODBUS protocol.
